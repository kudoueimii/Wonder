## 実行手順
お使いのPCにredisがインストールされていない場合はredisをインストールして起動

```
$ brew install redis
$ brew services start redis
```
インストール済みの方はここから
```
$ git clone git@github.com:kudoueimii/Wonder.git
$ cd wonder
$ bundle
$ rails db:create
$ rails db:migrate
$ rails db:seed
$ rails s
```
